# CliniSysCore (HPSR)

Sistema de agrupamento de módulos médicos do Hospital São Rafael (HPSR), com navegação lateral, carregamento de sistemas via iframe e interface padronizada para uso interno.

## Visão geral

O **CliniSysCore** foi criado para centralizar o acesso a diferentes sistemas e ferramentas médicas em uma única interface, facilitando a navegação e a organização operacional.

### Principais objetivos
- Centralizar módulos médicos em um único painel
- Padronizar a experiência visual
- Facilitar o acesso rápido aos sistemas
- Manter interface simples e responsiva para uso interno

## Funcionalidades

- Sidebar com navegação por módulos
- Botão para minimizar/expandir a barra lateral (mantendo apenas ícones)
- Persistência do estado da sidebar via `localStorage`
- Área de conteúdo com **iframe ocupando toda a região principal**
- Páginas institucionais (ex.: Diretoria, Parceiros)
- Layout com identidade visual do Hospital São Rafael (tema vinho/bege)

Este sistema foi desenvolvido para uso interno em ambiente de RP médico / gestão organizacional do Hospital São Rafael, com foco em organização, padronização e praticidade operacional.
